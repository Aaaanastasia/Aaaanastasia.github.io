# Anastasia Bogacheva — personal site

Personal brand site: analytics engineering, the systems I build, and my CV.

- **Live:** https://aaaanastasia.github.io
- **Stack:** static HTML + CSS + vanilla JS — no framework, no build step.
- **Design:** "Index" blueprint style — near-black surfaces, dashed containment
  borders, Archivo 900 display type, Geist Mono micro-labels, three rationed
  accents (periwinkle / amber / teal).
- **Art:** the contour background is generated, reproducibly (seed 42), by
  `tools/generate_art.py` (numpy · scipy · matplotlib).
- **CV:** `cv.html` mirrors `assets/Anastasia_Bogacheva_CV.docx`; the PDF is
  generated from the same document. The page is print-styled — printing it
  yields a clean black-on-white CV.

Built by hand — no template.
